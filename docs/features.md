# Features

## Multi-platform downloads

Download videos from TikTok, Instagram, YouTube Shorts and Twitter/X.

## HD quality

Videos are downloaded in high quality where the source allows.

## Watermark-free TikTok

TikTok videos are saved without watermarks for clean reuse.

## Fast and simple

No interface to learn — send a link, get the video file in chat.

## Runs inside Telegram

Everything happens in chat — there is no separate app or website to use.
